Privacy Policy

    Discard Tabs – Memory Saver

    Last updated: February 2026

    Overview

    Discard Tabs – Memory Saver is a Chrome extension designed to help users manually unload background browser tabs to reduce memory usage and improve browser performance.

    We respect your privacy. This extension is built to function entirely on your device and does not collect, transmit, or store personal data externally.

    Data Collection

    This extension:

    Does not collect personal information

    Does not track browsing history

    Does not transmit data to external servers

    Does not use analytics, cookies, or advertising technologies

    All functionality runs locally within the Chrome browser.

    Chrome Permissions

    The extension requests the following Chrome permissions:

    tabs

    Used solely to:

    Display a list of open tabs in the current browser window

    Discard (unload) selected background tabs when requested by the user

    No tab information is stored outside the browser or sent anywhere.

    storage

    Used only to:

    Save user interface preferences (such as dark mode) locally or via Chrome Sync

    This data remains within the user’s Chrome profile and is never accessed by the developer.

    Data Sharing

    Because the extension does not collect any user data, there is no data to share, sell, or transfer to third parties.

    Security

    All operations occur locally within the browser using official Chrome extension APIs.
    No external network connections are made by this extension.

    Changes to This Policy

    If the extension’s functionality changes in a way that affects privacy, this policy will be updated accordingly and the “Last updated” date will be revised.
