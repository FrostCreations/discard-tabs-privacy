Privacy Policy

Discard Tabs – Reduce Chrome RAM Usage Instantly
Last updated: February 2026

Overview

Discard Tabs – Reduce Chromium RAM Usage Instantly is a Chromium extension designed to help users manually unload background browser tabs to reduce memory usage and improve browser performance.

This extension operates entirely on your device and does not collect, transmit, or store personal data externally.

Data Collection

  • No personal information is collected.
  • No browsing history is tracked.
  • No data is transmitted to external servers.
  • No analytics, cookies, or advertising technologies are used.

Chrome Permissions

tabs

Used only to:

  • Display a list of open tabs in the current browser window
  • Discard selected background tabs when requested by the user

No tab data leaves the user’s browser.

storage

Used only to store user interface preferences such as dark mode locally or via Chrome Sync.

Data Sharing

Because no user data is collected, there is no data to sell, share, or transfer to third parties.

Security

All functionality runs locally within Chrome using official extension APIs. The extension makes no external network requests.

Changes to This Policy

If the extension’s functionality changes in a way that affects privacy, this policy will be updated and the date above will be revised.
